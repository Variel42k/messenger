# Инструкции по настройке клиентов мессенджера

## Веб-клиент

### Установка и запуск
1. Убедитесь, что у вас установлены Node.js и npm
2. Перейдите в директорию `messenger/web-client`
3. Установите зависимости: `npm install`
4. Запустите приложение: `npm start`
5. Откройте браузер и перейдите по адресу `http://localhost:3000`

### Функциональность
- Поддержка мультиязычности (английский, русский, испанский)
- Чат-интерфейс с возможностью отправки сообщений
- Раздел справки с инструкциями по настройке

## JavaFX-клиент

### Установка и запуск
1. Убедитесь, что у вас установлена Java 17 или выше
2. Убедитесь, что JavaFX библиотеки доступны в classpath
3. Перейдите в директорию `messenger/client`
4. Соберите проект: `mvn clean install`
5. Запустите приложение: `mvn javafx:run`

### Функциональность
- Поддержка мультиязычности (английский, русский, испанский)
- Чат-интерфейс с возможностью отправки сообщений
- Раздел справки с инструкциями по настройке
- Возможность выбора языка через меню "Language"

## Настройка подключения к серверу

Оба клиента должны быть настроены для подключения к серверу мессенджера:

- Адрес сервера: `http://localhost:8080` (по умолчанию)
- WebSocket endpoint: `ws://localhost:8080/ws` (по умолчанию)

Для изменения настроек подключения:
1. В веб-клиенте: измените конфигурацию в `src/config/api.js`
2. В JavaFX-клиенте: измените настройки в `src/main/resources/application.properties`

## Поддерживаемые языки

Приложение поддерживает три языка:
- Английский (en)
- Русский (ru)
- Испанский (es)

Переключение языка возможно в обоих клиентах:
- В веб-клиенте: через выпадающий список в шапке приложения
- В JavaFX-клиенте: через меню "Language" в главном меню

## Раздел справки

Оба клиента содержат раздел справки с инструкциями по настройке и использованию:
- В веб-клиенте: вкладка "Help" в навигационной панели
- В JavaFX-клиенте: пункт меню "Help" в главном меню
