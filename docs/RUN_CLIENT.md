# Запуск клиента мессенджера

Для удобного запуска клиента мессенджера без использования терминала предоставлены специальные скрипты.

## Требования

- Java 17 или выше

## Запуск на Windows

1. Запустите файл `messenger_client_run.bat` двойным щелчком мыши
2. Скрипт автоматически найдет установленную Java 17 или выше в системе
3. Если Java не найдена, скрипт предложит установить её
4. Приложение запустится в фоновом режиме

Поддерживаемые версии Java:
- Oracle JDK
- Eclipse Temurin (AdoptOpenJDK)
- Amazon Corretto
- Microsoft Build of OpenJDK

## Запуск на Linux/macOS

1. Убедитесь, что у вас установлена Java 17 или выше
2. Откройте терминал и перейдите в директорию client
3. Выполните команду:
   ```bash
   ./messenger_client_run.sh
   ```

## Альтернативный способ запуска

Вы также можете запустить клиент напрямую с помощью команды:
```bash
java -jar target/messenger-client-0.0.1-SNAPSHOT.jar
```

## Примечание

JAR-файл содержит все необходимые зависимости, включая JavaFX, поэтому дополнительная установка JavaFX не требуется.
